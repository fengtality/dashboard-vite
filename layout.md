# Hummingbot Dashboard - UI Layout Specification

## Design Decisions (Confirmed)

| Decision | Choice |
|----------|--------|
| Theme | Toggle (dark/light mode) |
| Sidebar | Collapsible to icons |
| Home Page | Yes - Dashboard with overall stats |
| Config Editor | Form builder (customizable) |
| Additional Pages | Settings page |
| Skipped | Market Data page |

---

## Pages Overview

### Implemented
1. **Connectors > [Exchange Name]** - Connector detail page (orders, positions, trades, balances)
2. **Connectors > Add Keys** - Manage API keys for exchange connectors
3. **Controllers > Grid Strike** - Custom config builder for Grid Strike strategy
4. **Bots > Active Bots** - Monitor running bots
5. **Bots > Deploy** - Pick controller â†’ multi-select configs â†’ deploy
6. **Bots > Archived Bots** - View archived bot performance

### Connector Detail Page Features
- **Spot Connectors**: Balances, Active Orders, Trades
- **Perpetual Connectors**: Balances, Active Orders, Positions, Trades, Funding Payments, Settings (position mode, leverage)

### Future Pages (not yet implemented)
- **Dashboard** (home) - Navigation cards to main sections
- **Portfolio > Current** - Current balances, distribution
- **Portfolio > History** - Historical portfolio value
- **Controllers > PMM Mister** - Custom config builder for PMM Mister strategy
- **Bots > Archived Detail** - Detailed analysis of archived bot
- **Settings** - API config, theme toggle

## shadcn/ui Components to Use

| Feature | Component(s) |
|---------|-------------|
| Sidebar navigation | `Sidebar`, `SidebarMenu`, `SidebarGroup`, `Collapsible` |
| Page headers | `Breadcrumb`, custom header component |
| Cards | `Card`, `CardHeader`, `CardContent`, `CardFooter` |
| Forms | `Form`, `Input`, `Select`, `Textarea`, `Label`, `Switch` |
| Tables | `Table` or `DataTable` (with sorting/filtering) |
| Buttons | `Button`, `ButtonGroup` |
| Dialogs | `Dialog`, `Sheet` (for side panels), `AlertDialog` |
| Feedback | `Toast` (via Sonner), `Badge`, `Progress` |
| Data display | `Tabs`, `Accordion`, `Skeleton` |
| Dropdowns | `DropdownMenu`, `Command` (for search) |
| Charts | `Chart` (Recharts-based) |
| Navigation | `Tabs`, `Pagination` |
